# 📚 Bookstore Management SQL Project

## 📌 Project Overview

This PostgreSQL project focuses on bookstore inventory management, customer orders, and sales analysis using SQL queries.
The project demonstrates database design, data analysis, and business problem-solving using PostgreSQL.

---

# 🛠️ Tools Used

* PostgreSQL
* pgAdmin
* CSV Files
* GitHub

---

# 🗂️ Database Tables

## 1. Books

Contains book details such as:

* Book ID
* Title
* Author
* Genre
* Published Year
* Price
* Stock

## 2. Customers

Contains customer information such as:

* Customer ID
*  Name
*  email
*  phone
*  City
* Country
  

## 3. Orders

Contains order transaction details such as:

* Order_ID
* Customer_ID
* Book_ID
* Order_Date
* Quantity
* Total_Amount
  

---

# 📊 Business Problems Solved

### Basic SQL Queries

* Retrieve Fiction genre books
* Find books published after 1950
* Identify customers from Canada
* Analyze November 2023 orders
* Calculate total stock available
* Find the most expensive book
* Identify customers ordering multiple quantities
* Retrieve high-value orders above $20
* List all book genres
* Detect low-stock books
* Calculate total revenue

### Advanced SQL Queries

* Analyze books sold by genre
* Calculate average Fantasy book prices
* Identify repeat customers
* Find the most frequently ordered book
* Retrieve top 3 expensive Fantasy books
* Analyze author-wise sales
* Locate high-spending customers by city
* Identify highest spending customer
* Calculate remaining inventory after orders
